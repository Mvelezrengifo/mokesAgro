# mokesAgro 🌱

Plataforma digital orientada al sector agro, con arquitectura completa
frontend + backend, desarrollada como MVP funcional a nivel de código.

## Estado del proyecto
El sistema cuenta con la lógica completa de backend y frontend.
No se realizó el empaquetado final como aplicación debido a decisiones
externas al desarrollo técnico.

## Funcionalidades
- Comunicación frontend ↔ backend
- Arquitectura modular
- Base preparada para despliegue o empaquetado como app

## Tecnologías
- Node.js
- JavaScript
- API REST
- Frontend web

## Nota
El proyecto se presenta como muestra técnica y arquitectónica
dentro del ecosistema **MOKES**.
