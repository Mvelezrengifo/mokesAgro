mokesAgro 🌱
Plataforma digital orientada al sector agro, con arquitectura completa frontend + backend, desarrollada como MVP funcional a nivel de código.

Estado del proyecto
El sistema cuenta con la lógica completa de backend y frontend. No se realizó el embalaje final como aplicación debido a decisiones externas al desarrollo técnico.

Funcionalidades
Frontend de comunicación ↔ backend
Arquitectura modular
Base preparada para desplegar o empaquetar como aplicación
Tecnologías
Node.js
JavaScript
API REST
Interfaz web
Nota
El proyecto se presenta como muestra técnica y arquitectónica dentro del ecosistema MOKES .
